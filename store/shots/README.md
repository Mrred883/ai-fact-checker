# Landing-page screenshots

Drop three PNGs here to populate the homepage screenshot row. Until they exist,
the homepage shows styled placeholders in their place.

Expected filenames (exact):

- `feed.png` — the Feed tab with verdict cards
- `assets.png` — the Assets tab with the AI-origin meter
- `sentiment.png` — a Sentiment result card

Generate them with `store/screenshot-framer.html` (pick each preset, upload the
matching popup screenshot, download), then rename the downloads to the names
above and place them in this folder. Any width works; they scale to fit.
